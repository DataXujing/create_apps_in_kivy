+ 完全基于Python,快速实现Windows,Linux, MacOs, ios, Android跨平台GUI，NUI APP开发
+ 融合机器学习，深度学习，计算机视觉项目实战

适合人群：

+ 有一定的C++,Python编程基础，对GUI开发有基础的了解
